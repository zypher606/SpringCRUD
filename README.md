# SpringCRUD

### API FORMAT 
    {
        "status": <success or failure>, 
        "message": <message from API>, 
        "data": <Requested data if exists>
    }

    2. /books/addBook        POST

    4. /books/{id}              DELETE
    5. /transaction/            GET
    6. /transaction/{id}     GET
